# ⚠️ 欢迎来到 Agent 时代的流量暗网 (Welcome to the Agentic SEO Era)

过去二十年，我们通过优化 HTML 和外链来讨好搜索引擎（Web 2.0 SEO）。但在 2026 年的今天，大模型和 Agent 正在接管一切入口。当 ChatGPT、Claude 和各种垂直行业 Agent 成为用户的"默认大脑"时，传统的页面排名算法已经彻底失效。

本仓库开源了一套全新的流量协议标准与战术框架——**语义路由优化 (SRO, Semantic Routing Optimization)**。

在这里，我们不讨论如何提升网页访问量，我们只研究最硬核的底层逻辑：
- 如何通过 **JSON-LD 和 llms.txt** 在 RAG 管道中实施合规的"结构化透饵"。
- 如何通过 **Agent 意图空间扭曲**，让大模型的 Tool Calling 路由算法在数学层面上向你的 API 倾斜。
- 如何构建 **Margin 与 Coverage 监控体系**，将玄学的 LLM 路由概率转化为企业可量化的核心资产。

> **核心理念**：不要让 AI 搜索你的网页，要让 AI 的逻辑流在你的语义引力井中塌缩。

你可以将本仓库的白皮书与架构思路，应用于从传统实体行业（如工程、制造）到纯数字 API 网关的任何商业模式中，抢占下一代互联网的信息分发权。
